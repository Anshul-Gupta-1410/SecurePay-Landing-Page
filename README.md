# SecurePay Landing Experience

A next-generation fintech landing website designed with a dark, futuristic aesthetic, utilizing glassmorphism, 3D visuals, and dynamic layout animations. 

Built for The OLB Group, Inc.

## Features
- **Sleek Aesthetic**: Custom "Kinetic Void" dark mode with Electric Blue (`#00D1FF`) and Deep Violet (`#7000FF`) accents.
- **Glassmorphism**: Elegant translucent panels, layered borders, and cohesive `blur` filters.
- **3D Interactive Orbit Nodes**: Revolving security nodes and abstract financial visuals driven by Framer Motion math.
- **Bento Grids**: Engaging feature breakdowns covering SecureGuard, Robust APIs, and Automated Billing.
- **Content Expansion**: A "Markets & Solutions" dashboard preview with expanding UI segments.
- **Partner Displays**: Floating, interactive typographic integration of QuickBooks, Stripe, PayPal, and more.

## Tech Stack
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Language**: TypeScript
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animation Engine**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: Lucide React & Optimized Next/Image SVGs & PNGs

## Getting Started

1. Navigate into the Next.js directory:
   ```bash
   cd securepay
   ```

2. Install the necessary dependencies:
   ```bash
   npm install
   ```

3. Spin up the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to experience the landing page.

## Deployment
This project is fully ready to be deployed on platforms like Vercel or Netlify. Since it leverages Next.js, deploying to Vercel provides out-of-the-box configuration. You just need to link this repository to a Vercel standard project targeting the `/securepay` root directory.
